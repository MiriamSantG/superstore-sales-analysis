# Superstore Sales Analysis — EDA

## Overview
Exploratory Data Analysis on 4 years of retail sales data (10K orders).

## Key Findings
- Tables & Bookcases generate negative profit margins
- Higher discounts consistently destroy profit margin
- West region leads in sales and profitability
- Home Office segment achieves highest profit margin (14%)

## Tools
Python, Pandas, Matplotlib, Seaborn
